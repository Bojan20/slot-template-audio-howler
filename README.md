# slot-template-audio-howler
Repository for Audio Team to place audio assets

Usage: please alter the settings.json to conform to the needs of your project.

To copy the audio files from this repo into your game repo, run the copyAudio.js script in this folder.  The script reads in configuration params from the adjacent settings.json file so make sure and update those with the correct relative paths to your game project for your game.

To run the script, type:

$node copyAudio.js
